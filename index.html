Basic Git Commands

  • git --version
  • git config --global user.name (a one time thing)
  • git config --global user.email (a one time thing)
  • git init (to initialize an empty git repository)
  • ls (to show files and directories that are not hidden) Other Similar Commands
  • git status
  • git add <file name>
  • git add <file-1> <file-1> <file-1> (to add multiple files)
  • git add . (to add all files)
  • git commit -m "message"
  • git log (to know the history of commits)
